# DocTrack v2 — база знаний

## Как пользоваться
- Любая задача по коду → product-and-code.md + style-and-patterns.md
  (оба файла небольшие, читать целиком, а не выборочно по разделам)
- Любой list/table/kanban view → дополнительно docs/clients-reference.tsx
  (полный UI-референс страницы Клиенты — читать перед созданием)
- Стандарт DoD и сдачи задачи → не здесь; см. корневой CLAUDE.md,
  раздел «Сдача результата»

## Файлы
- product-and-code.md — что за продукт, для кого, sidebar-навигация,
  текущий пользователь, стек, aliases, карта папок
- style-and-patterns.md — code style, единый источник данных kanban/table,
  UI-конвенции (язык, цвета, границы, layout), команды проверок
